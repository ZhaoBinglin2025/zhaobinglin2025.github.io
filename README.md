## ✨ 为什么选择这个模板？

虽然基于经典的 *Minimal Mistakes* 主题，但本仓库经过了彻底的“现代化”重构，具备以下核心优势：

- **🚀 极致极简设计**：采用现代排版（Inter/PT Serif），专注于留白与可读性。所有核心内容均采用完美的居中布局（Flexbox），在大屏与小屏上均有极佳平衡感。
- **🧊 灵动导航/UI**：拥有现代化的玻璃拟态（Glassmorphism）导航栏，具备毛玻璃模糊效果，并在滚动时自动吸顶。
- **📄 沉浸式论文展示**：针对学术成果优化的单列布局。内置独有的 **Metadata Chips** 系统，支持自动展示影响因子（IF）、导师（Advisor）、论文类型等标签，信息层级分明。
- **🖼️ 自动化灯箱（Lightbox）**：内置零配置的图片展示系统。无论是博客还是爱好页面的图片，点击即可放大，支持键盘左右切换和 Esc 取消，提供极致的视觉交互。
- **📱 响应式优化**：针对移动端体验进行了深度调优，解决了传统主题在手机上图片过大或排版错乱的问题。
- **💡 AI-Native 架构**：代码结构经过专门优化，更易于被 AI 代理理解。你可以通过自然语言直接命令 AI 修改网站布局或添加内容。



## 🛠 快速上手 (Fork & Build)

1. **Fork 本仓库**：点击右上角的 `Fork` 按钮将其复制到USERNAME/USERNAME.github.io，其中USERNAME是你的github用户名。
2. **个性化配置**：打开 `_config.yml` 文件，更新 `owner` 部分的信息（姓名、学校、邮箱、简介等）。
3. **极速部署**：进入仓库设置 (`Settings`) -> `Pages`，选择从 `main` 分支部署，你的个人主页将在几分钟内上线。
4. **AI 维护**：阅读 [**Antigravity 指南：零代码掌控网页开发**](./ANTIGRAVITY_GUIDE.md)，学习如何使用 AI 代理来维护你的网站。



## 🤖 开启“AI 原生”维护模式

**你不需要成为前端工程师，也能拥有世界上最漂亮的个人主页。**

通过使用 **Antigravity** 这样的智能代理，你只需用中文描述你的想法，它就能在几秒钟内完成修改。本仓库已将所有样式变量集中化，确保 AI 修改时绝不会出错。

### 💡 实战 Prompt (提示词) 示例：

- **新增论文**：
  > “我有一篇新论文被 CVPR 录用了，信息如下：[粘贴论文信息]。请帮我把它添加到 `publications.md` 的顶部。用翡翠绿色的标签标注其影响因子，并附上 PDF 链接。”
- **修改全局配色**：
  > “我想把的全站的主题色从蓝色改成更加沉稳的‘斯坦福红’（#8C1515）。请更新 CSS 变量，确保悬浮状态和背景色在白色主题下依然协调。”
- **优化手机端展示**：
  > “我觉得手机端打开时，‘个人介绍’部分的图片太占空间了。请帮我写一段响应式代码，让图片在窄屏下只占 30% 宽度并右对齐。”
